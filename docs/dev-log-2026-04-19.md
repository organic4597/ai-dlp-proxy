## 4. iptables nf_tables 문법 오류 수정

**문제**: 라즈베리파이 Ubuntu의 `iptables v1.8.10 (nf_tables)` 환경에서 startup 시 오류 발생

```
iptables: unexpected ! flag before --match
[WARN] OUTPUT REDIRECT 실패 — xt_owner 모듈 확인 필요
```

**원인**: iptables legacy 문법 vs nf_tables 문법 차이

```bash
# legacy 문법 (실패)
iptables -t nat -A OUTPUT -p tcp --dport 443 ! -m owner --uid-owner 0 -j REDIRECT

# nf_tables 문법 (수정)
iptables -t nat -A OUTPUT -p tcp --dport 443 -m owner ! --uid-owner 0 -j REDIRECT
```

**수정 파일**: `setup_transparent.sh` (삭제 규칙 포함 총 2군데 수정)

**결과**: OUTPUT REDIRECT 규칙 정상 적용 확인
```
4  REDIRECT  tcp  --  anywhere  anywhere  tcp dpt:https ! owner UID match root redir ports 4443
```
